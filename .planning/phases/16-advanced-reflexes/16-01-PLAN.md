    1. `CircuitBreakerState` frozen dataclass:
       - `agent_id: str`
       - `consecutive_block: int = 0` (count of consecutive BLOCK-level actions)
       - `consecutive_observe: int = 0` (count of consecutive OBSERVE actions after open)
       - `is_open: bool = False`

    2. `update_circuit_state(state: CircuitBreakerState, mode: ResponseMode) -> CircuitBreakerState`:
       Pure function. If mode == BLOCK: increment consecutive_block, reset consecutive_observe.

       If consecutive_block >= 5: set is_open = True.
       If consecutive_observe >= 10 and is_open: set is_open = False.
       Returns new immutable state.

    3. `evaluate_circuit_breaker(state: CircuitBreakerState) -> ReflexResult`:
       If state.is_open: return ReflexResult(allow=True, reflex_kind="circuit_breaker",
         inject_message=f"[SOMA] Agent {state.agent_id} quarantined -- sustained high pressure",
         detail=f"consecutive_block={state.consecutive_block}, trust -> 0.1")
       Else: return ReflexResult(allow=True)

    Import ReflexResult from soma.reflexes. Import ResponseMode from soma.types.

    Tests in `tests/test_graph_reflexes.py`: 8 tests covering all behavior cases above.
